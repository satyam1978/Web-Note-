IP Address (IPv4)

IPv4 is 32-bit, written in dotted decimal (e.g., 172.217.16.142). Total addresses: ~4.3 billion.

Public vs Private IP

Type Description
Public IP Globally unique, assigned by ISP, used for Internet communication.
Private IP Used inside local networks only. Common ranges:
  • 10.0.0.0 â€“ 10.255.255.255
  • 172.16.0.0 â€“ 172.31.255.255

IPv6

IPv6 uses 128-bit addresses, written in hexadecimal with colons. Example:

2001:0db8:85a3:0000:0000:8a2e:0370:7334

Supports ~340 undecillion unique addresses.

MAC Address

A permanent hardware ID, 48 bits, shown in hex (e.g., 3C:22:FB:A3:B4:C5).

Finding MAC Address

  • Windows: Run ipconfig /all or getmac /v.
  • macOS: Run ifconfig in Terminal.

DNS â€” Domain Name System

DNS translates human-readable domains to IP addresses.

Steps

  1. Check local caches (browser, OS, router).
  2. Ask recursive resolver (ISP or public DNS).
  3. Root servers point to TLD servers.
  4. TLD servers point to authoritative name servers.
  5. Authoritative server returns the final IP.

Finally, the browser connects to the IP on the correct port (e.g., 443 for HTTPS).

Short Summary

  • Internet = networks connected via routers.
  • IPv4 = 32-bit; IPv6 = 128-bit.
  • MAC addresses = hardware IDs.
  • Ports = service identifiers.
  • DNS = translates names into IP addresses.
