HTML Media â€” Introduction

Web pages are more than just text and images. To make them interactive and engaging, you need to embed audio and video in a standardized, plugin-free way.

Browsers now support native HTML5 <video> and <audio> tags for this purpose.

1: Video Tag Basics

The simplest video tag:

<video src="my-awesome-video.mp4"></video>

Add controls for a functional player:

<video src="my-awesome-video.mp4" controls></video>

You can add width, height, autoplay, loop, and poster attributes:

<video src="video.mp4" width="640" height="360" controls autoplay muted loop poster="images/video-thumbnail.jpg"></video>

2: Audio Tag

Embed audio files with the <audio> tag:

<p>Listen to our theme song:</p>
<audio src="audio/theme-song.mp3" controls></audio>

Supports the same attributes as <video>: autoplay, loop, muted.
